# Bash Automation – System Info Script

## Beschreibung
Dieses Skript sammelt grundlegende Systeminformationen
und gibt sie übersichtlich im Terminal aus.

## Funktionen
- Anzeige von Hostname
- Aktueller Benutzer
- Datum und Uhrzeit
- Uptime
- Festplattennutzung
- Arbeitsspeicher

## Nutzung
```bash
chmod +x system-info.sh
./system-info.sh
